## Current state

- `DOCUMENT_REGISTRY.csv` contains 173 `pending` rows across 21 domains.
- `ATLAS_ADOPTION_MATRIX.csv` assigns every row one K-01 disposition.
- `MAP_TO_EXISTING` and `MERGE` rows must use the named Repository A owner.
- `ADOPT` and `DEFER` rows require explicit future Founder review.
- No row becomes canonical, approved, generated, or implementation-ready through registration or disposition alone.
- Legacy Library Atlas files with old `ATLAS-*` IDs are `ARCHIVE_REFERENCE / NONCANONICAL`.

The Atlas validators check structural consistency and safety. Passing validation does not approve document content or authorize generation, WordPress, runtime, import, publishing, deployment, production, or Phase 1 AI.
