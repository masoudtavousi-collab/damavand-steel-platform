1. Copy `.env.example` to `.env` and replace every placeholder locally.
2. Review version constraints and run `composer install` when dependency installation is approved.
3. Provision a database and configure the web root as `public/`.
4. Install WooCommerce, Blocksy, Blocksy Pro, Elementor, and Elementor Pro through the approved package channel.
5. Set WordPress language to Persian (`fa_IR`), timezone to `Asia/Tehran`, and verify RTL output.
6. Configure inquiry workflows only after their architecture and privacy controls are approved.

This repository deliberately contains no plugin implementation or commercial package binaries.
