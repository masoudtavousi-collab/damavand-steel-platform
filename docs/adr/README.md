Create records as `NNNN-short-title.md` using `0000-template.md`. ADRs are immutable after acceptance; supersede them with a new record instead of rewriting history.

## Governing Relationships

- ADRs must reference the applicable [Core Project Principles](../00_PROJECT_BIBLE.md#core-project-principles).
- Architecture decisions must reference [Enterprise Architecture](../02_ARCHITECTURE.md).
- Business-impacting decisions must reference [Business Rules](../03_BUSINESS_RULES.md).
- Technology-impacting decisions must reference [Technology Stack](../05_TECH_STACK.md).
- The [Decision Log](../10_DECISION_LOG.md) indexes decisions but does not replace the ADR as the authoritative decision record.
